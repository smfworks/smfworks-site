# WisdomForge Audio Series: Epictetus — The Stoic Way

## About This Series

This is the first WisdomForge audio philosophy series. Each chapter adapts the life and teachings of Epictetus for a specific age group, read by George — a warm, captivating storyteller voice.

The series is built from the four consumer booklets:
- **Ages 5–10:** *The Boy with No Name* — simple, gentle, story-based
- **Ages 11–14:** *The Boy with No Name* — respectful, slightly more mature
- **Ages 15–18:** *The Boy with a Borrowed Name* — philosophical, emotionally honest
- **Adult:** *The Examined Life* — direct, reflective, no self-help padding

Each booklet shares the same six chapter themes. Each audio chapter runs approximately 8–10 minutes.

- **Voice:** George — Warm, Captivating Storyteller

- **Total runtime:** ~24 clips × 8–10 minutes ≈ 3.5–4 hours
- **Music:** Optional intro/outro theme available at `/audio/epictetus/theme.mp3`

## Usage

These recordings are intended for:
- Direct web playback in the WisdomForge library
- Download for offline listening
- Integration into podcast feeds
- Accompaniment to the written booklets
