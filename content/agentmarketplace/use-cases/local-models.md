## Top picks

### Ollama + Open WebUI
Simplest way to download, run, and chat with local models on macOS, Linux, or Windows.

### llama.cpp
Best for squeezing performance out of consumer hardware and custom quantization.

### Jan
Clean desktop app for local AI with model management and built-in conversations.

### LM Studio
User-friendly model loader with good GPU offload and chat/completion modes.

## How to choose

- Use Ollama for quick setup and terminal-first workflows.
- Use llama.cpp for maximum control and edge deployment.
- Use Jan for a polished local desktop experience.
- Use LM Studio when you want a GUI but still run everything locally.

## Common gotchas

- Quantization trades quality for speed; test on your actual prompts.
- Local agents need a machine with enough VRAM or unified memory.
- Keep model files and logs on encrypted storage if you handle sensitive data.
