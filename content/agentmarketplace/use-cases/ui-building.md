## Top picks

### v0
Best for converting a text prompt or screenshot into a polished React/Tailwind component.

### Bolt.new
Runs the full stack in the browser. Best for rapid prototyping with backend + frontend + deploy.

### Lovable
Strong for Figma-to-code and iterative visual editing with AI-assisted revisions.

### Replit Agent
Good for education and quick full-stack apps with built-in hosting.

## How to choose

- Use v0 when you already have a design system and need clean component output.
- Use Bolt.new when you need a live deployable prototype in minutes.
- Use Lovable when design fidelity and visual iteration matter most.
- Use Replit Agent for teaching, learning, or one-off experiments.

## Common gotchas

- Generated UI often needs accessibility and responsive cleanup.
- Don’t let agents pick dependencies freely; pin versions and audit bundles.
- Split complex UIs into component prompts rather than one giant prompt.
