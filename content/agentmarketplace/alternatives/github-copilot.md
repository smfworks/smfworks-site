# Alternatives to GitHub Copilot

Top alternatives for different needs:

- **Cursor** — editor-native agent with multi-file edits and composer mode.
- **Claude Code** — terminal-first agent for complex refactor and debugging.
- **Continue** — open-source, local-first assistant for VS Code and JetBrains.
- **Codeium / Windsurf** — free tier, fast autocomplete, cascade agent.
- **Tabnine** — enterprise-focused, privacy-first completions.

Use the [agent directory](/agentmarketplace/agents) to compare pricing and features side-by-side.
