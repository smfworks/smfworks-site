# Alternatives to Cursor

- **GitHub Copilot** — deep IDE integration and broad autocomplete.
- **Zed** — fast native editor with built-in agentic editing.
- **Claude Code** — terminal agent that works with any editor via MCP.
- **Windsurf** — Codeium's agentic IDE with cascade workflows.
- **Cline** — open-source VS Code agent with local model support.
