# Deploy Cline in VS Code

1. Open VS Code Extensions and search **"Cline"**.
2. Install the Cline extension.
3. Add your API key (OpenAI, Anthropic, Google, DeepSeek, or local OpenAI-compatible endpoint).
4. Optional: enable MCP servers for extended tools.
5. Start a task from the Cline panel.

Source: https://github.com/cline/cline
