# Vector Database

Managed vector database for RAG, agent memory, and semantic search at scale.

**Provider:** [Pinecone](https://www.pinecone.io)
**Pricing:** Usage-based, from ~$0.10 / GB-hour
**Best for:** Agents that need long-term memory, document retrieval, or semantic search

Pinecone is a fully managed vector database with metadata filtering, hybrid search, and no index tuning. It plugs into LangChain, LlamaIndex, and custom agent loops.
