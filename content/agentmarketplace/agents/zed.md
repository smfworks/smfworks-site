Zed is a Rust-built, GPU-accelerated code editor from the creators of Atom and Tree-sitter. It pairs real-time collaborative editing with first-class AI agent support, and its Zeta2.1 model predicts edits with fewer tokens and lower latency. The editor is free and open source; AI features run on hosted credits or your own API keys.
