Microsoft Scout is an always-on, autonomous AI agent that runs on Windows 11 and macOS. It connects to Teams, Outlook, OneDrive, SharePoint, and the local file system, then plans and executes multi-step tasks while pausing for human approval before consequential actions. Currently available through the Microsoft Frontier early-access program, it signals the shift from chat-based assistants to background workplace agents.
